# Resumo de Aprendizado: CSS Fundamental

## Por que usar CSS Externo?
O CSS serve para dar estilo (cor, tamanho, posição) ao HTML. Usar um arquivo externo (como `style.css`) é melhor porque mantém o código organizado, facilita a manutenção e permite que você use o mesmo estilo em várias páginas diferentes.

## O Modelo de Caixa (Box Model)
Esta é a parte vital para posicionar elementos:
* **Padding (Preenchimento):** O espaço interno, entre o conteúdo e a borda.
* **Margin (Margem):** O espaço externo, que empurra outros elementos para longe.



## Glossário de Propriedades
* `color`: Altera a cor do texto.
* `background-color`: Define a cor de fundo.
* `margin`: Espaço por fora do elemento.
* `padding`: Espaço por dentro do elemento.
* `display: flex`: Alinha os itens de forma dinâmica e fácil.

## O Poder das Classes
As **classes** (ex: `.minha-caixa`) servem para identificar grupos de elementos. Em vez de estilizar todos os `p` (parágrafos) iguais, eu crio uma classe e aplico apenas onde eu quero, mantendo o código limpo.

# biblio-finder
O projeto Recomendação de Livro tem como objetivo auxiliar pessoas a encontrarem livros de forma rápida, prática e eficiente, de acordo com seus gostos e preferências literárias. A proposta busca facilitar a escolha de leituras por meio da indicação de obras organizadas por gêneros, como romance, fantasia, suspense, terror, autoajuda, entre outros.
